## Overview

Clean, gradient-based breathing app showing different breathing phases as distinct screen states.

## Key Screens

### Phase Screens
Three distinct screens for breathing phases:

1. **"Breathe In..."**
   - Green/teal gradient background
   - Circular progress indicator
   - Timer display

2. **"Paused"**
   - Yellow/orange gradient background
   - Pause state indicator
   - Hand gesture icon

3. **"Breathe out..."**
   - Orange/red gradient background
   - Circular progress indicator
   - Timer display

## Design Elements

### Color Strategy
Each breathing phase has a distinct color:
- **Inhale**: Cool colors (teal, blue-green)
- **Hold/Pause**: Warm neutral (yellow, amber)
- **Exhale**: Warm colors (orange, coral)

This creates instant visual recognition of the current phase.

### Gradient Backgrounds
- Full-screen gradients
- Smooth color transitions
- Creates immersive feeling
- Changes with breathing phase

### Progress Indicator
- Circular ring around content
- Shows progress through current phase
- Clean, minimal design

### Typography
- Large, clear phase text
- Ellipsis (...) indicates ongoing action
- Timer in smaller text below

## Animation Concept

The design implies:
- Smooth transitions between phase colors
- Circular progress animation
- Possibly morphing gradients
